A) Desciption - top

Kurosaki HTML5 Audio Player is written in Pure JavaScript. No JavaScript library needed. It uses HTML5 <audio> tag, and adds its own controls to every tag its used on. If HTML5 <audio> tag has controls attribute specified, Kurosaki HTML5 Audio Player removes this attribute, and adds its own controls.

C) Options - top

Default Options Are:

To know more about every option and its possible values, read further.

1) selector - top

selector option accepts CSS selector. Selector should target HTML5 <audio> tags.

2) width - top

width option accepts desired width of player. It accepts values in percents, pixels, and ems.

3) maxWidth - top

maxWidth option accepts desired maximum width of player. It accepts values in percents, pixels, and ems.

4) minWidth - top

minWidth option accepts desired manimum width of player. It accepts values in percents, pixels, and ems.

5) theme - top

theme option accepts player theme name. Themes available are: 'light', 'dark', 'red', 'green', 'blue', 'yellow'.

6) seekBar - top

seekBar option determines whether or not player seek bar will be shown. It accepts two values: true or false.

7) volumeControl - top

volumeControl option determines whether or not player volume control will be shown. It accepts two values: true or false.

Note: To see working player instantiated using different combinations of settings see demo.
